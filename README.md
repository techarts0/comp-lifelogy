## 💡 项目简介 (Introduction)

你是否觉得现代生活充满了焦虑、内耗和不确定性？
也许问题不在于你不够努力，而在于你运行的“认知操作系统”已经过时了。

**《计算生活学》** 不是一本教你写代码的技术书，而是一本**“认知升级手册”**。我尝试将自己在计算机行业 25 年的从业经验，沉淀为 12 个核心思维模型。

我想告诉你：**计算机科学不仅是关于机器的学科，更是关于如何在这个复杂的、充满不确定性的世界中，建立秩序、管理资源、应对变化的顶级智慧。**

这里没有晦涩的公式，只有从硬核技术中提炼出的生活哲学。

## 📚 目录 (Table of Contents)

本书共 12 章，对应代码江湖的 12 种心法：

- [ ] **01. 包容 (Inclusive)** - 眼里揉不进沙子吗 [*(噪声容限)*]
- [ ] **02. 定力 (Equanimity)** - 其实被动也挺好的 [*(中断 vs 轮询)*]
- [ ] **03. 洞察 (Insight)** - 不畏浮云遮望眼 [*(抽象与建模)*]
- [ ] **04. 务实 (Pragmatism)** - 莫在浮沙起高楼 [*(晶体管可靠性)*]
- [ ] **05. 取舍 (Trade-offs)** - 世间安得双全法 [*(CAP定理)*]
- [ ] **06. 放下 (Letting Go)** - 健忘才会快了 [*(GC与存储分级)*]
- [ ] **07. 规则 (Discipline)** - 自由源于自律 [*(协议与SOP)*]
- [ ] **08. 分寸 (Boundaries)** - 封装自我，明确接口边界 [*(封装与接口)*]
- [ ] **09. 机巧 (Leverage)** - 苦干蛮干不如巧干 [*(算法与复用)*]
- [ ] **10. 接纳 (Acceptance)** - 我们都是裱糊匠 [*(不可判定性)*]
- [ ] **11. 余地 (Redundancy)** - 晴带雨伞，饱带饥粮 [*(备份与高可用)*]
- [ ] **12. 慷慨 (Generosity)** - 赠人玫瑰，手有余香 [*(Open Source)*]

*(点击链接可阅读对应章节初稿，`[ ]` 代表待更新，`[x]` 代表已发布)*

## 🤝 如何参与 (Contributing)

这本书本身就是一个**开源项目**。我非常欢迎你通过以下方式参与：

1.  **阅读与反馈**：如果你对某一章有感触，或者发现了逻辑漏洞，欢迎提交 [Issue](https://github.com/techarts0/comp-lifelogy/issues) 讨论。

3.  **纠错 (Pull Requests)**：发现错别字或技术描述不准确？直接 Fork 并提交 PR 吧！
